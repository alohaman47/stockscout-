# 📡 StockScout — สแกนหาหุ้นเทรนด์ชัด หลายตลาด

ระบบสแกนหุ้นอัตโนมัติ สไตล์เดียวกับ ZhuLinsen (fork → รันบน GitHub Actions → ส่ง Telegram, ต้นทุนศูนย์) แต่ **ให้คะแนนด้วยคณิตศาสตร์ที่ตรวจสอบได้** ไม่ใช่ความเห็น LLM

หาหุ้นที่ **เทรนด์ขึ้น/ลงชัดและสะอาด** เหมาะกับการถือยาววันถึงเดือน รองรับหลายตลาดพร้อมกัน (US, ไทย SET, ฮ่องกง, จีน, ญี่ปุ่น, เกาหลี, ไต้หวัน) ผ่าน yfinance ตัวเดียว ไม่ต้องมี API key

## คะแนนคำนวณยังไง

แต่ละตัวให้คะแนน 4 มิติ:

1. **ทิศทาง + ความสะอาด (แกนหลัก)** — เอา log ราค 90 วันมาลากเส้น regression: `score = ความชันรายปี × R²` ยิ่ง R² เข้าใกล้ 1 ราคายิ่งเกาะเส้นตรง = เทรนด์สวย ถือสบาย ถ้าเหวี่ยงมั่ว R² ต่ำ คะแนนหด
2. **MA stack (Minervini template)** — ราคา/SMA50/150/200 เรียงถูกทิศไหม (0–1)
3. **ADX** — เทรนด์มีแรงจริง ไม่ใช่ไซด์เวย์
4. **จังหวะเข้า** — RSI + ระยะห่างจากไฮ 20 วัน บอกว่า "โซนย่อน่าเข้า" หรือ "ยืดเกินรอย่อ"

เกณฑ์ผ่าน (ปรับได้): R² ≥ 0.55, ADX ≥ 18, template ≥ 0.70 → จัดอันดับเอา Top ขาขึ้น + Top ขาลง

## 🦋 โหมด Butterfly Effect

3 เลนส์เสริมที่มองข้าม "เทรนด์ที่วิ่งแล้ว":

- **จุดชนวน (Coiled Spring)** — หาหุ้นที่กำลังขดตัว: Bollinger squeeze (BBW percentile ต่ำ) + ATR percentile ต่ำ + NR7 + วอลุ่มแห้ง รวมเป็น `Coil score` 0–1 สูง=บีบแน่นรอระเบิด พร้อมทิศที่เอียง (↑/↓) จับเทรนด์*ก่อนเกิด*
- **แรงกระเพื่อม (Ripple)** — วิเคราะห์ lead-lag cross-correlation ทั้ง watchlist หาคู่ "ตัวนำ → ตัวตาม" ที่ตัวตามมักขยับตามหลัง 1–5 วัน ถ้าตัวนำเพิ่งขยับ ≥2% จะติดธง 🔥 ให้จับตาตัวตาม *(ความสัมพันธ์รายวันอ่อนและไม่เสถียร ใช้เป็น watch-hint เท่านั้น)*
- **ความโกลาหล (Hurst)** — Hurst exponent ต่อตัว: >0.5 เทรนด์มีแนวโน้มไปต่อ, <0.5 มีแนวโน้มเด้งกลับ (mean-revert), ~0.5 สุ่ม แสดงเป็นคอลัมน์ 🦋H ในตารางหลัก จับคู่กับ R²: R² บอกเทรนด์*สะอาด*, Hurst บอกเทรนด์*จะอยู่ต่อไหม*

ปรับได้ผ่าน Variables: `COIL_MIN`, `RIPPLE_LAG`, `RIPPLE_MINCORR`

## ติดตั้ง (2 นาที ทำบนเว็บล้วน ไม่ต้องต่อ Telegram)

**1. Fork repo นี้** (ปุ่มมุมขวาบน)

**2. เปิด Actions** — แท็บ `Actions` → กด "I understand my workflows, go ahead and enable them"

**3. รันครั้งแรก** — เลือก `StockScout Daily` → `Run workflow`

**4. ดูรายงาน** — ได้ 2 ทาง:
- **หน้า Actions:** คลิกเข้าไปในรอบรันที่เพิ่งเสร็จ → เลื่อนลงเห็น **Summary** เป็นตารางสวย ๆ ทันที
- **ไฟล์ในrepo:** เปิด `reports/latest.md` (อัปเดตทุกครั้งที่รัน) หรือ `reports/YYYY-MM-DD.md` ดูย้อนหลัง

**(ทางเลือก) ปรับเกณฑ์** ที่ `Settings → Secrets and variables → Actions → Variables` — `TOP_N`, `MIN_R2`, `MIN_ADX`, `MIN_TMPL`

**(ทางเลือกอนาคต) Telegram** — ถ้าวันไหนอยากได้แจ้งเตือนเข้ามือถือ ค่อยเพิ่ม Secrets `TELEGRAM_BOT_TOKEN` + `TELEGRAM_CHAT_ID` โค้ดรองรับไว้แล้ว ไม่ใส่ก็ไม่ทำงาน

## แก้ watchlist

เปิดไฟล์ `watchlist.txt` บนเว็บ GitHub กด edit ได้เลย — ใส่ ticker บรรทัดละตัว, `#` = คอมเมนต์
suffix ต่อตลาด: ไทย `.BK` · ฮ่องกง `.HK` · จีน `.SS/.SZ` · ญี่ปุ่น `.T` · เกาหลี `.KS` · ไต้หวัน `.TW` · US ไม่ต้องใส่
